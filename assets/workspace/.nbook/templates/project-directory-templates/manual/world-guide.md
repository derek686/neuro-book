# 世界导览

本文件是玩家和主持人都可以阅读的世界观导览。它只写开局时适合公开的信息，隐藏真相和未揭示秘密应留在 `lorebook/` 或主持侧说明中。

## 基调

描述本 Project 的题材、氛围、叙事尺度和默认玩法体验。

## 世界一览

用短段落说明这个世界是什么、玩家通常会在哪里开始、常见冲突来自哪里。

## 地域

列出玩家开局可能知道的主要地域。稳定细节链接到 `lorebook/location/...` 或 `lorebook/world/...`。

## 势力

列出玩家开局可能知道的主要势力。稳定细节链接到 `lorebook/faction/...`。

## 常识

记录普通角色在开局时合理知道的常识。不要写只有主持或隐藏角色知道的信息。

## 禁忌与剧透边界

记录玩家手册中不应提前揭示的主题、秘密或真相类型。需要揭示时由主持人和玩家确认。
